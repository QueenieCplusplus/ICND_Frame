# ICND Frame
資料框的內容解讀 (根據 IEEE 802.3 和 802.2 標準)

* MAC sublayer (IEEE 802.3)


      +----------+----------+-----------+--------+----------------+-----+
      | Preamble | Dest Add | Sorce Add | Length | Variable (Data)| FCS |
      +----------+----------+-----------+--------+----------------+-----+
      /* 第一欄位在於通知第二欄位接收者：『 哎喔！資料框進來囉～～～ 』*/
